Installation Pre-Requisites

1. SimplePie (http://simplepie.org/) installed in ./simplepie/n.n[.n]/
2. YUI 2.x (http://developer.yahoo.com/yui/2/) installed in ./yui/n.n[.n]/
3. jQuery installed in ./jquery/n.n.n
4. jQuery BackStretch installed in ./jquery-backstretch/n.n[.n]

... where n.n[.n] is the version number of the installed packages.
